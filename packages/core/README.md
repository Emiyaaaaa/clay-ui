# @kopi-ui/core

Core skeuomorphic React components for Kopi UI.

## Installation

```bash
pnpm add @kopi-ui/core
```

## Usage

```tsx
import { Button } from '@kopi-ui/core';

export function App() {
  return <Button label="Click me" />;
}
```

## Theming

Fully customizable via Tailwind CSS v4 tokens and utilities.
