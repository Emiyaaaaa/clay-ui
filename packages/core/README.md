# @clay-ui/core

Core skeuomorphic React components for Clay UI.

## Installation

```bash
pnpm add @clay-ui/core
```

## Usage

```tsx
import { Button } from '@clay-ui/core';

export function App() {
  return <Button label="Click me" />;
}
```

## Theming

Fully customizable via Tailwind CSS v4 tokens and utilities.
